# Trailing Lambda

When the last parameter of a function is a function type, you can use trailing lambda syntax. The lambda is written outside the parentheses, making the code more concise and natural.

If the function has only one parameter and it is a function type, you can even omit the parentheses entirely and write the lambda directly.

Trailing lambda syntax is widely used in collection operations, callbacks, and DSL construction. It makes code read more like natural language.
