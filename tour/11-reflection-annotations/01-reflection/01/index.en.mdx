# Reflection

Reflection allows programs to inspect type information at runtime and dynamically access members. Cangjie provides reflection capabilities through the `TypeInfo` type, which is the core type for reflection and records metadata for any type.

There are three ways to obtain `TypeInfo`: through an instance object, through a generic parameter, or through a type name string. `TypeInfo.of(obj)` returns the runtime type information of an object, while `TypeInfo.of<T>()` returns static type information for type `T`. For class types, `TypeInfo.of(obj)` returns `ClassTypeInfo`, which provides more class-specific reflection interfaces.

Reflection can access public members of a type, including static variables, instance variables, properties, and methods. Through `getStaticVariable` and `getInstanceVariable`, you can find member variables by name, then use `getValue` and `setValue` to read and write their values.

<Note>Reflection can only access `public` members; `private` and `protected` members are not visible.</Note>

Try modifying the parameter of `setValue` and observe the output changes.

CJ:
import std.reflect.*

// Define a simple counter class
public class Counter {
    public static var step: Int64 = 2      // Static variable
    public var value: Int64 = 10           // Instance variable
}

main(): Unit {
    // Create an object instance
    let counter = Counter()
    
    // Get type information of the object
    let info = ClassTypeInfo.of(counter)
    println("Type name: ${info.name}")
    println()
    
    // Find member variables by name
    let valueInfo = info.getInstanceVariable("value")
    let stepInfo = info.getStaticVariable("step")
    
    // Read initial values
    println("Before modification:")
    println("  value = ${(valueInfo.getValue(counter) as Int64).getOrThrow()}")
    println("  step = ${(stepInfo.getValue() as Int64).getOrThrow()}")
    println()
    
    // Modify variable values through reflection
    valueInfo.setValue(counter, 25)
    stepInfo.setValue(5)
    
    // Read modified values
    println("After modification:")
    println("  value = ${(valueInfo.getValue(counter) as Int64).getOrThrow()}")
    println("  step = ${(stepInfo.getValue() as Int64).getOrThrow()}")
}
