# Tuples

Tuples are lightweight data structures that group multiple values of potentially different types. They are created using parentheses with elements separated by commas.

A tuple's type is determined by its elements' types. For example, `(Int64, String)` is a tuple type containing an integer and a string. Elements can be of different types.

Access tuple elements using bracket indexing starting from `0`. You can also use destructuring to bind tuple elements to separate variables.

Tuples are ideal when you need to return multiple values or temporarily group related values without defining a dedicated struct.
