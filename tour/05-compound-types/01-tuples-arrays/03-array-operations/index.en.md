# Array Operations

Cangjie arrays provide a rich set of operations. The `append` method adds elements to the end, and the `size` property returns the array length.

Arrays support slicing with `arr[start..end]` to get sub-arrays. You can also modify elements of a mutable array by index.

Common array operations include iteration (`for-in`) and searching. Arrays are reference types, so passing an array passes a reference.

The example demonstrates common array operations including appending, modifying, and slicing.
