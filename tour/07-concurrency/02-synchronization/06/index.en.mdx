# Where to Go from Here...

The [Cangjie Documentation](https://cangjie-lang.cn/docs) is a great place to start. It contains the language specification, standard library reference, and guides.

To learn how to organize and work with Cangjie code, read the [cjpm package manager guide](https://cangjie-lang.cn/docs) in the Tools section of the documentation.

If you need help with the standard library, see the [stdlib API reference](https://cangjie-lang.cn/docs?url=/1.0.5/libs/std/std_module_overview.html).

You might also continue with the extended modules in this tour — Packages and Tooling, FFI, Macros, and Reflection — available in the sidebar.
