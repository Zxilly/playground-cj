# Where to Go Next

Congratulations! You have completed the core part of the Cangjie Tour and mastered the basic concepts of the language and concurrent programming.

To continue learning Cangjie in depth, you can refer to the following resources:

- [Cangjie Official Documentation](https://cangjie-lang.cn/docs) contains language specifications, standard library reference, and programming guides - the most authoritative learning materials
- [cjpm Package Manager Guide](https://cangjie-lang.cn/docs) introduces how to organize and manage Cangjie projects, suitable for actual development
- [Cangjie Standard Library API Reference](https://cangjie-lang.cn/docs?url=/1.0.5/libs/std/std_module_overview.html) provides detailed documentation for all standard library modules

You can also continue exploring the extension modules of this tutorial—Packages and Toolchain, FFI, Macros, and Reflection—which can be found in the sidebar.

The best way to learn is through practice. Try writing a small project in Cangjie to solve a real problem, and deepen your understanding through practice.

Happy coding!
