# 接下来去哪里

祝贺你！你已经完成了仓颉 Tour 的核心部分，掌握了这门语言的基础概念和并发编程。

要继续深入学习仓颉，可以参考以下资源：

- [仓颉官方文档](https://cangjie-lang.cn/docs) 包含语言规范、标准库参考和编程指南，是最权威的学习材料
- [cjpm 包管理器指南](https://cangjie-lang.cn/docs) 介绍如何组织和管理仓颉项目，适合实际开发
- [仓颉标准库 API 参考](https://cangjie-lang.cn/docs?url=/1.0.5/libs/std/std_module_overview.html) 提供了所有标准库模块的详细文档

你也可以继续探索本教程的扩展模块——包与工具链、FFI、宏和反射——在侧边栏中可以找到它们。

最好的学习方式是实践。尝试用仓颉编写一个小项目，解决一个实际问题，在实践中深化理解。

祝编程愉快！
