# Generic Structs

In addition to generic functions, Cangjie supports generic types. A struct can be parameterized with type parameters, which is useful for reusable data structures.

`Pair<T, U>` stores two values of potentially different types. Methods on the struct return the exact types the caller specified.

Change `Pair<String, Int64>` to `Pair<Bool, Float64>` and update the constructor arguments.
