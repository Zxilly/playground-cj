# Annotations

Cangjie supports **annotations**, marked with the `@` symbol. Annotations attach metadata to code elements for use by compilers or frameworks.

Common uses of annotations:
- Marking test methods
- Controlling serialization behavior
- Providing metadata for frameworks

Cangjie includes some built-in annotations, and you can also define custom ones. Annotations don't change the runtime behavior of code, but can be read and processed by tools and frameworks.
