# Known Languages

Cangjie will feel familiar if you know Java, Go, Kotlin, or Swift. It uses similar concepts — functions, classes, generics, and interfaces — with its own syntax and a few new ideas like pattern matching and lightweight threads.

If you already know one of those languages, select it in <Highlight target="langpicker">the language toggles</Highlight> in the header. Comparison tabs will appear throughout the tour showing how the same concept looks in Cangjie versus the language you know.

If you don't have a language background, just read straight through — the tour is self-contained.
