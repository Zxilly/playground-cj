# The Playground

This tour is built atop the Cangjie Playground, a web service that runs on remote servers.

The service receives a Cangjie program, compiles, links, and runs it inside a sandbox, then returns the output.

There are limitations to programs that can run in the playground:

- Execution time and memory usage are capped.

Visit the [Cangjie Playground](https://cangjie-lang.cn/playground) to try it directly in your browser.
