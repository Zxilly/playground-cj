# Playground

The code examples in this tutorial run on the Cangjie Playground—an online compilation environment hosted on remote servers.

When you click the run button, the code is sent to the server where it's compiled, linked, and executed in a sandbox environment, then the output is returned and displayed. This allows you to start learning immediately without configuring a local development environment.

You can also visit the [Cangjie Playground](https://cangjie-lang.cn/playground) directly to experiment freely with various code snippets.

<Note>To ensure service stability, programs in the Playground have limits on execution time and memory usage. Complex or long-running programs may not execute properly.</Note>
