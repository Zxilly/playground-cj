# Cangjie Offline (Optional)

This tour is also available as a stand-alone program you can use without internet access. It builds and runs the code samples on your own machine.

To run Cangjie locally, first [download and install the compiler](https://cangjie-lang.cn/download), then compile programs with `cjc`:

```
cjc hello.cj -o hello && ./hello
```

Of course, you can continue to take the tour through this web site.
