# Hello World

Every Cangjie program starts execution from the `main` function. `main()` is the entry point of the program and is called automatically when the program starts.

The `println()` function prints content to the console and adds a newline at the end. You can also use `print()` to output content without a trailing newline.

The `main` function in Cangjie does not require an explicit return type. The function body is enclosed in curly braces `{}` and contains the code to execute.
