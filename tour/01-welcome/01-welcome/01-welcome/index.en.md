# Welcome to the Cangjie Tour

Welcome to the interactive tour of the Cangjie programming language!

Cangjie is a modern, general-purpose programming language that combines static typing, type inference, pattern matching, and more into a concise yet expressive syntax. It is suitable for system programming, application development, and many other scenarios.

In this interactive tutorial, you will learn the core concepts of Cangjie through a series of progressive examples. Each page contains editable, runnable code samples that you can modify and execute directly in your browser.

Click the "Run" button to run your first Cangjie program!
