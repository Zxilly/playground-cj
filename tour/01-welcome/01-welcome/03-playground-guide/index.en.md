# Playground Guide

This Playground is your interactive environment for learning Cangjie. Here are some tips:

**Editor**: The code editor is on the left side. You can freely edit the code. The editor provides syntax highlighting to help you read code more easily.

**Run Button**: Click the Run button to execute the current code. The output will be displayed in the output panel.

**Navigation**: Use the "Previous" and "Next" buttons at the bottom to navigate between tutorial sections. The table of contents on the left lets you jump to any section quickly.

**Experiment**: Don't be afraid to modify the example code! You can always edit and re-run the code -- this is the best way to learn programming.
