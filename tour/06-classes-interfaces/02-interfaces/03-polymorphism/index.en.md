# Polymorphism

Polymorphism is one of the core features of object-oriented programming. Through interface or base class types, different objects can be handled uniformly.

When a function parameter type is an interface, any type implementing that interface can be passed as an argument. At runtime, the method corresponding to the object's actual type is called.

Polymorphism makes code more flexible and extensible -- adding new types only requires implementing the relevant interface, without modifying code that uses the interface.

The example demonstrates runtime polymorphism through an array of interface-typed values.
