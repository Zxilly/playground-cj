# Subtyping

In Cangjie, if class B inherits from class A, or class B implements interface A, then B is a subtype of A. Values of a subtype can be used wherever the parent type is expected.

Interfaces can also have subtype relationships. One interface can extend another, forming an interface inheritance hierarchy.

Subtyping is the foundation of Cangjie's type system. It allows writing generic code that handles different types sharing common characteristics while maintaining type safety.
