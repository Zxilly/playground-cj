# Access Rules

An extension lives outside the original type definition, so it can only access the type's public members.

```cangjie
extend Rectangle {
    func area(): Float64 {
        return this.width * this.height
    }
}
```

It cannot reach private or internal fields. All four methods here (`area`, `perimeter`, `isSquare`, `describe`) use only the public `width` and `height`.

Try adding a private field to `Rectangle` and accessing it from the extension -- the compiler will reject it.
