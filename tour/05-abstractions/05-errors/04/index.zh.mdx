# 自定义异常

通过继承 `Exception` 或其子类，可以创建自定义异常类型。自定义异常可以携带额外的字段，比如错误码、字段名等，为错误处理提供更丰富的上下文信息。

定义自定义异常时，需要：
- 使用 `open class` 声明（允许进一步继承）
- 调用父类构造函数 `super(message)` 传递错误消息
- 重写 `getClassName()` 方法返回异常名称

右侧代码定义了两个异常：`ValidationError` 携带字段名，`NetworkError` 携带状态码。在 `catch` 块中可以访问这些额外信息。
