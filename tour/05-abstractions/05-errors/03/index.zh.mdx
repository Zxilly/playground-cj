# finally 块

`finally` 块无论 `try` 块是正常结束、抛出异常还是提前返回，都会被执行。它通常用于释放资源、关闭文件、清理状态等"善后"工作。

执行顺序是：`try` 块 → `catch` 块（如果抛出异常且被捕获）→ `finally` 块。即使 `catch` 块中有 `return` 语句，`finally` 也会在返回之前执行。
