# 类的定义

`class` 定义引用类型。与 `struct` 不同，类的实例在赋值或传参时不会复制数据——两个变量将指向同一个对象。这种行为使得类适合表示需要共享或可变的数据。

类使用 `class` 关键字定义，可以包含成员变量、构造函数、成员函数等。成员变量默认必须初始化或在构造函数中赋值。使用 `let` 声明不可变成员，使用 `var` 声明可变成员。

下方的 `Counter` 类定义了两个成员变量和一个成员函数。注意将 `counter` 赋值给 `sameCounter` 后，修改 `sameCounter` 的字段也会影响 `counter`，因为两者指向同一对象。

<Note>
类的所有实例成员变量必须在构造函数中完成初始化，否则编译器会报错。
</Note>

<CompareWith lang="java">

<CompareWith lang="go">
Go 没有类的概念，使用 struct 和方法组合实现面向对象。仓颉的 class 更接近传统 OOP 语言。
</CompareWith>

<CompareWith lang="rust">
Rust 的 struct 默认是值类型，需要用引用或包装类型实现共享。仓颉的 class 天生是引用类型。
</CompareWith>
