# Classes

A `class` defines a reference type. Assigning a class instance to another variable does not copy the data -- both variables point to the same object.

In this example, `cat` and `sameCat` refer to the same `Animal`. Changing `sameCat.name` also changes `cat.name`.

Try changing `sameCat.name` to a different value and verify that `cat.name` reflects the change.
