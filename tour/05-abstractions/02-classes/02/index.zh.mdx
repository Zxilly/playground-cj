# 构造器

类使用 `init` 定义构造器。一个类可以定义多个构造器，只要它们的参数列表不同（构成重载）。构造器的函数体中必须完成所有未初始化实例成员变量的赋值。

仓颉还支持主构造器（primary constructor），它的名字与类名相同。主构造器的参数列表中可以用 `let` 或 `var` 声明成员变量形参，同时完成成员变量定义和构造函数参数声明的功能。

下方的 `Person` 类演示了两种构造器定义方式。第一个 `init` 是普通构造器，第二个是主构造器的简化写法。两种方式创建的对象行为完全一致。

<Note>
如果类没有自定义构造器且所有成员变量都有初始值，编译器会自动生成一个无参构造器。
</Note>
