# 属性

`prop` 通过 `get()` 和可选的 `set(v)` 定义计算属性。对外使用时像普通字段，但读写在背后执行自定义逻辑。属性可以实现访问控制、数据验证、延迟计算等功能。

不带 `mut` 修饰符的属性只能定义 `get()`，是只读属性。带 `mut` 的属性必须同时定义 `get()` 和 `set(v)`，是可读写属性。计算属性不占用额外存储空间，它们基于其他数据动态计算。

下方的 `Temperature` 类将摄氏度作为实际存储字段，华氏度通过计算属性推导。当摄氏度改变时，华氏度自动更新——因为它在每次访问时重新计算。

<Note>
计算属性在 getter 和 setter 中访问自身会造成无限递归，这与函数递归同理。
</Note>
