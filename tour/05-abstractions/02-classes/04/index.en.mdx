# Inheritance

A class marked `open` can be inherited. Use `<:` (not `:`) to extend a parent class, and call `super(...)` to invoke the parent constructor.

In this example, `Dog` and `Cat` inherit from `Animal` and override `speak()`. Both can be stored in `Array<Animal>`.

Try adding a `Fish` subclass that returns `"blub"` from `speak()`.
