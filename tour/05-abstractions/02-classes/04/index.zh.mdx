# 继承

标记为 `open` 的类才能被继承。使用 `<:` 符号（注意不是 `:`）声明继承关系。子类构造器必须通过 `super(...)` 调用父类构造器，完成父类成员的初始化。

仓颉是单继承：一个类只能有一个直接父类。继承形成了"is-a"关系——子类对象可以当作父类使用。这使得我们可以用 `Array<父类>` 同时存储多种子类对象，运行时自动调用正确的实现（多态）。

下方的 `Dog` 和 `Cat` 继承自 `Animal`，都重写了 `speak()` 方法。两个子类对象存入同一个数组，遍历时各自表现不同行为。

<Note>
子类构造器必须调用父类构造器。如果未显式调用 `super(...)`，编译器会尝试调用父类的无参构造器。
</Note>
