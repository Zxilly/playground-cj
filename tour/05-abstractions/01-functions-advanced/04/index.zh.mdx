# 嵌套函数

函数内部可以定义其他函数，这称为嵌套函数或局部函数。嵌套函数的作用域仅限于外层函数内部，对外不可见，但可以被外层函数返回后在更大范围内使用。

嵌套函数可以访问外层函数的局部变量和参数。这使得外层函数可以将复杂逻辑拆分成多个内部辅助函数，同时不污染外部命名空间。

右侧示例中，`processNumbers` 内部定义了两个嵌套函数 `sum` 和 `average`，它们都访问参数 `numbers`。外层函数调用这两个嵌套函数并返回结果。这是函数式编程中常见的模式。

<Note>
嵌套函数与 Lambda 的主要区别：嵌套函数有名字，可递归调用，语法更接近普通函数；Lambda 更简洁，适合简单的单次使用场景。
</Note>

<CompareGroup>

<CompareWith lang="c">
C 不支持嵌套函数，无法在函数内部定义其他函数。仓颉的嵌套函数提高了代码组织性。
</CompareWith>

<CompareWith lang="java">
Java 支持局部类和匿名类，可以在方法内定义，但语法较为冗长。仓颉的嵌套函数更简洁。
</CompareWith>

<CompareWith lang="go">
Go 不支持在函数内定义命名函数，只能使用匿名函数。仓颉的嵌套函数提供了更好的可读性和可递归性。
</CompareWith>

<CompareWith lang="rust">
Rust 支持嵌套函数，语法与仓颉类似。嵌套函数同样可以捕获外层变量形成闭包。
</CompareWith>
