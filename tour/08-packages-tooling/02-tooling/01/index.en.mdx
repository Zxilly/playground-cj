# cjc Compiler

`cjc` is the compiler command for the Cangjie programming language, used to compile Cangjie source code into executable files.

The most basic usage is to directly compile a single file. Without any options, `cjc` compiles the source file into an executable named `main`. You can also use the `-o` option to specify the output filename, for example `cjc hello.cj -o hello`.

`cjc` also supports compiling entire package directories. Use the `-p` option to specify the package path, for example `cjc -p src/` will compile all source files under the `src/` directory.

<Note>`cjc` automatically performs frontend compilation, backend compilation, and linking, without requiring manual linker invocation.</Note>
