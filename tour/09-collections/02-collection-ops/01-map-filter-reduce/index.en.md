# map/filter/reduce

Functional collection operations are powerful tools for processing data. `map` applies a transformation to each element, `filter` selects elements by condition, and `reduce` combines all elements into a single value.

These operations can be chained together, creating clear data processing pipelines. Compared to manually writing loops, functional operations are more concise and expressive.

These three operations are the cornerstones of functional programming, and mastering them will greatly improve your ability to work with collection data.
