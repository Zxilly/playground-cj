# Collection Conversions

Different collection types can be converted between each other. Common conversions include turning an array into an `ArrayList`, converting an `ArrayList` to a `HashSet` for deduplication, or building a `HashMap` from a collection.

Conversion operations let you choose the data structure best suited for the current need. For example, use `ArrayList` for collecting data, convert to `HashSet` for deduplication, and build a `HashMap` for lookups.

Using collection conversions wisely makes your code more efficient while keeping it concise.
