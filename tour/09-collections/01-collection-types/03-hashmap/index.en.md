# HashMap

`HashMap` is a key-value collection that provides fast lookup of values by key. Keys must be hashable types, while values can be any type.

Common operations include `put` to add or update key-value pairs, subscript `map[key]` to get values, and `contains` to check if a key exists. If you `put` the same key multiple times, the later value overwrites the earlier one.

`HashMap` is a fundamental data structure for implementing common programming patterns like lookup tables, counters, and caches.
