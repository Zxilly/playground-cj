# Collection Iteration

All collection types support iteration using `for-in` loops. For `ArrayList` and `HashSet`, iteration yields elements directly; for `HashMap`, iteration yields key-value tuples.

In addition to basic `for-in` loops, you can use indexed iteration to get both the element and its position.

Iteration is the most fundamental operation for processing collection data. Mastering iteration patterns for different collection types is the foundation for working with collections effectively.
