# Tokens

`Tokens` is a sequence of `Token` objects representing a fragment of source code. Each `Token` has a `kind` (like `TokenKind.IDENTIFIER` or `TokenKind.INTEGER_LITERAL`) and a string `value`.

You can construct tokens with `Token(kind, value)` and combine them with `+`.

Change `"1"` to `"42"` in the first token and run the program to see the updated output.
