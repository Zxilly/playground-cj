# 宏的声明与参数

宏函数声明为 `public macro Name(input: Tokens): Tokens`。宏必须声明为 `public`，因为它们总是被其他包使用。参数类型和返回类型都是 `Tokens`，表示 Token 序列。

宏可以分为非属性宏和属性宏两种。非属性宏只接受一个参数，即被修饰的代码片段，调用时使用 `@MacroName(...)`，作用于声明时可省略括号。属性宏接受两个参数：第一个是 `[]` 中的属性，第二个是被修饰的代码，调用时使用 `@MacroName[attr](...)`。

非属性宏调用时，括号内的内容（或声明本身）成为 `input` 参数。属性宏调用时，`[]` 内的内容成为第一个参数 `attr`，后面修饰的代码成为第二个参数 `input`。宏返回的 `Tokens` 会替换源代码中的整个宏调用。

右侧示例中，`AddPrefix` 是一个属性宏，接收属性和输入两部分，将它们拼接后返回。当调用 `@AddPrefix[public ]struct Data` 时，`attr` 是 `"public "`，`input` 是 `struct Data {...}`，返回值是 `public struct Data {...}`。

<Note>属性宏的中括号和内容是强制的，即使为空也必须写 `@MacroName[](...)`。</Note>
