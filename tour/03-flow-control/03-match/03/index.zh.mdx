# 绑定模式

绑定模式不仅匹配值，还将匹配到的值绑定到一个变量上，这样你可以在分支中使用这个值。语法是 `case 变量名 =>`，变量名可以是你想要的任何标识符。

绑定模式通常用于匹配并处理剩余情况。与通配符 `_` 不同，绑定模式让你能访问匹配到的值。`case other =>` 会匹配任何值，并将值绑定到 `other` 变量。

<Note>绑定模式不能与 `|` 一起使用，因为每个模式需要独立绑定变量，无法共享。</Note>
