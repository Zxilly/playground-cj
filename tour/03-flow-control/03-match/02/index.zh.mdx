# 值模式与多值匹配

值模式是最简单的模式，它匹配等于某个字面量的值。你可以使用整数、字符串、字符等字面量作为模式。

使用 `|` 可以在一个 `case` 中匹配多个值，这在处理多个相关值时非常方便。`case 3 | 4 | 5` 表示匹配 3、4 或 5 中的任意一个。

`_` 是通配符模式，可以匹配任何值。它通常作为最后一个分支，用于处理所有其他情况，确保匹配的穷尽性。
