# Exercise: Pattern Matching

As a way to practice pattern matching, implement a `classify` function that describes an `Option<Int64>` value.

Given an `Option<Int64>`, return a `String`:

- `Some(n)` where `n > 0` → `"positive"`
- `Some(0)` → `"zero"`
- `Some(n)` where `n < 0` → `"negative"`
- `None` → `"none"`

Implement this in the `func classify` provided.
