# While 循环

`while` 循环在条件为真时反复执行代码块。它先检查条件，再决定是否执行循环体。如果条件一开始就为假，循环体一次也不会执行。

`do-while` 循环则先执行一次循环体，再检查条件。这意味着循环体至少会被执行一次，即使条件一开始就为假。

与许多语言类似，`while` 和 `do-while` 的条件必须是布尔类型表达式。在循环体中，通常需要修改与条件相关的变量，否则可能导致无限循环。

</CompareWith>

<CompareWith lang="rust">

</CompareWith>
