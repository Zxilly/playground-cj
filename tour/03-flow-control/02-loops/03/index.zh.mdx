# 区间

区间用于表示一个有序的整数序列。仓颉有两种区间：左闭右开区间 `start..end` 不包含 end，左闭右闭区间 `start..=end` 包含 end。

可以用 `: step` 指定步长。例如 `0..10 : 2` 表示从 0 开始，每次加 2，直到小于 10。步长可以是负数，用于递减序列。当步长为负数时，start 应该大于 end。

步长不能为 0，否则会编译错误。如果步长与区间的方向矛盾（如步长为正但 start > end），则产生空区间——遍历时循环体一次也不会执行。

<CompareGroup>

<CompareWith lang="c">
C 没有区间类型，需要用 `for (int i = 0; i < 10; i++)` 的形式手动控制。仓颉的区间语法更简洁声明式。
</CompareWith>

<CompareWith lang="java">
Java 也没有区间类型，需要手写循环。Java 的 IntStream 可以生成范围序列，但不如仓颉的区间语法简洁。
</CompareWith>

<CompareWith lang="go">
Go 没有内置区间类型，需用 `for i := 0; i < 10; i++` 手动管理循环变量。仓颉的区间语法 `0..10` 更声明式。
</CompareWith>

<CompareWith lang="rust">
Rust 的 `0..10` 是左闭右开区间，`0..=10` 是闭区间。仓颉采用完全相同的语法，Rust 用户会感到熟悉。
</CompareWith>

</CompareGroup>
