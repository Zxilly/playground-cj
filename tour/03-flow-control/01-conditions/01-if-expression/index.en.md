# if Expression

The `if` expression is the most basic conditional control structure in Cangjie. It selects different code branches based on a boolean condition.

The basic syntax is `if (condition) { branch1 } else { branch2 }`. The condition must be of type `Bool`.

The `else` branch is optional. You can also use `else if` for multi-level conditional checks.

Unlike C, Cangjie does not allow integers or other non-boolean types as conditions.
