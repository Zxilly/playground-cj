# if as Expression

In Cangjie, `if` is not just a control statement -- it is an expression that produces a value.

When an `if` expression is used for assignment, the value of the last expression in each branch becomes that branch's return value. The types of both branches must be compatible.

This is similar to the ternary operator in other languages, but Cangjie's `if` expression is more flexible as branches can contain multiple statements.
