# if 语句

仓颉的 `if` 语句用于根据条件执行不同的代码分支。条件表达式的值必须是 `Bool` 类型，当条件为 `true` 时执行 `if` 分支，为 `false` 时执行 `else` 分支。

与许多语言不同，仓颉要求条件必须用括号 `()` 包围，花括号 `{}` 也是必需的，即使分支内只有一行代码。这些设计避免了常见的编码错误，提高代码可读性。

仓颉不允许用整数或其他类型作为条件。在 C 或 JavaScript 中，`if (number)` 会根据 number 是否为 0 来决定分支，但在仓颉中这会编译报错——条件必须是明确的布尔表达式。
