# if 表达式

在仓颉中，`if` 不仅是语句，还可以作为表达式使用——它能返回一个值。当一个含 `else` 分支的 `if` 表达式赋值给变量时，每个分支代码块的最后一个表达式就是该分支的返回值。

所有分支必须返回相同类型的值。如果某个分支返回字符串，另一个分支返回整数，编译器会报错——这确保了 `if` 表达式总是有确定的类型。

如果 `if` 表达式没有赋值给变量，它的类型为 `Unit`，此时不需要关心各分支的返回类型。不含 `else` 的 `if` 表达式类型也为 `Unit`，因为它可能不会执行任何分支。

右侧代码展示了将 `if` 表达式的结果直接赋值给变量的用法。这种方式比先声明变量再用 `if` 赋值更加简洁。试着修改 `temperature` 的值，看看 `weather` 会变成什么。

</CompareWith>

</CompareWith>
