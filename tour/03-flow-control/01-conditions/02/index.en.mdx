# if Expressions

In Cangjie, `if` is not only a statement but can also be used as an expression - it can return a value. When an `if` expression with an `else` branch is assigned to a variable, the last expression in each branch's code block becomes that branch's return value.

All branches must return values of the same type. If one branch returns a string and another returns an integer, the compiler will report an error - this ensures that `if` expressions always have a definite type.

If an `if` expression is not assigned to a variable, its type is `Unit`, and there's no need to worry about the return types of each branch. An `if` expression without an `else` also has type `Unit`, since it might not execute any branch.
