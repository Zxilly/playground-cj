# 带数据的枚举

枚举构造器可以携带参数，不同构造器的参数类型和数量可以不同。这使枚举能够表达更丰富的数据结构。

参数在构造器名后的括号中声明，如 `Circle(Float64)` 携带一个 Float64 参数，`Rectangle(Float64, Float64)` 携带两个。

在 `match` 表达式中，可以通过模式匹配提取构造器携带的数据。`case Circle(r) =>` 将圆的半径绑定到变量 `r`，在 `=>` 后的代码中可以直接使用。
