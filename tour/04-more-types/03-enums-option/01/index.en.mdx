# Enum Types

An `enum` type defines a type by enumerating all possible values, called constructors. Constructors are separated by `|` and can be either parameterless or carry parameters.

To construct an enum value, use the form `TypeName.Constructor`, such as `Direction.North`. You can also use the constructor name directly when there is no naming conflict.

The `match` expression is used to handle all possible cases of an enum. Each `case` branch matches one constructor, with the handling code following `=>`. `match` requires exhaustive coverage of all possibilities, and the wildcard `_` can be used as a fallback.
