# 数组切片

使用 Range 类型可以在下标中一次性获取某一段元素，得到一个新的数组。Range 语法为 `start..end`，表示从 start 开始到 end 结束（不包含 end）。

在下标中可以省略 Range 的起始或结束位置。省略起始位置时，从索引 0 开始；省略结束位置时，延续到数组末尾。切片操作返回的是新数组，修改切片不会影响原数组。

按索引遍历时，可以使用 `for (i in 0..arr.size)` 配合索引访问，也可以使用 `for ((i, v) in arr.enumerate())` 同时获取索引和值。

<Note>切片范围 `start..end` 中，start 和 end 都不能超过数组长度。如果 start 大于 end 或出现负数，会触发运行时错误。</Note>

<CompareWith lang="python">

</CompareWith>

<CompareWith lang="go">

</CompareWith>
