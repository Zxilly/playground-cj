# 数组

`Array<T>` 是相同类型元素的固定大小序列，其中 T 表示元素类型。数组使用方括号 `[]` 创建字面量，编译器会自动推断元素类型。

数组元素通过索引访问，索引从 0 开始。`.size` 属性返回元素个数。使用 `for-in` 循环可以按顺序遍历数组中的所有元素。

数组虽然不能改变长度（不能添加或删除元素），但可以修改已有位置的元素值。需要注意的是，Array 是引用类型——赋值或传参时不会拷贝元素数据，多个引用指向同一份数据。

<Note>访问数组时，索引不能为负数或超过 `size - 1`，否则会在编译时或运行时报错。</Note>

<CompareGroup>

<CompareWith lang="c">
C 的数组是固定大小的值类型，不能动态调整长度。C 通常用指针进行数组操作，更容易出错。
</CompareWith>

<CompareWith lang="java">
Java 的数组也是固定大小，但数组本身是对象（引用类型）。Java 的 ArrayList 提供动态数组功能。
</CompareWith>

<CompareWith lang="go">
Go 的数组是值类型，赋值会拷贝整个数组。仓颉的 Array 是引用类型，赋值只拷贝引用，所有引用共享同一份数据。
</CompareWith>

<CompareWith lang="rust">
Rust 的 `Vec<T>` 是可变长度的动态数组，而仓颉的 `Array<T>` 长度固定。仓颉还有 `ArrayList<T>` 提供动态扩容能力。
</CompareWith>

</CompareGroup>
