# 构造器与值语义

`struct` 是值类型。将结构体赋值给另一个变量会创建独立的副本，修改副本不会影响原始值。

一个 `struct` 可以定义多个 `init` 构造器（需要参数不同），也可以定义与类型名相同的主构造器。主构造器的参数可以用 `let` 或 `var` 修饰，同时定义字段和构造参数。

右侧代码展示了构造器重载和值语义拷贝。`Rectangle` 有两个 `init` 构造器，分别创建矩形和正方形。变量 `b` 是 `a` 的独立副本，修改 `b` 不会影响 `a`。

<Note>
定义构造器时，必须初始化所有未设置初值的实例成员变量，否则会编译报错。
</Note>

<CompareGroup>

<CompareWith lang="c">
C 的结构体没有构造器概念，需要手动初始化字段。赋值时按值拷贝整个结构体。
</CompareWith>

<CompareWith lang="java">
Java 用构造方法初始化对象，可以定义多个构造方法（重载）。赋值时是引用复制，不是值拷贝。
</CompareWith>

<CompareWith lang="go">
Go 的 struct 没有构造器语法，通常定义 `NewXxx()` 函数来创建实例。赋值时也会拷贝。
</CompareWith>

<CompareWith lang="rust">
Rust 的 struct 可以有多个 `new` 方法，赋值也会拷贝（除非被移动）。
</CompareWith>

</CompareGroup>
