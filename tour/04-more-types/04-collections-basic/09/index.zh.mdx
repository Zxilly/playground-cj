# 练习：词频统计

实现一个函数 `wordCount`，统计输入字符串数组中每个单词的出现次数，返回 `HashMap<String, Int64>`。

右侧代码已提供函数签名和测试数据，你只需填充函数体。提示：遍历数组，对每个单词，如果 `HashMap` 中已有就递增计数，否则插入初始值 1。这与上一页的标签统计逻辑相同。

完成后点击运行，查看统计结果是否正确。

实现思路：
1. 创建空的 `HashMap<String, Int64>`
2. 遍历输入数组中的每个单词
3. 对每个单词，判断是否已在 map 中
4. 如果存在，递增计数；否则插入初始值 1
5. 返回填充好的 `HashMap`

<CompareWith lang="c">

<CompareWith lang="java">
Java 的实现与仓颉类似，可以使用 `Map.computeIfAbsent()` 或 `putIfAbsent()` 一行代码完成计数。

<CompareWith lang="go">

</CompareWith>

<CompareWith lang="rust">
Rust 可以用 `counts.entry(word).and_modify(|c| *c += 1).or_insert(1)` 一行代码实现。初学者可以用更清晰的 if-else 结构。
</CompareWith>
