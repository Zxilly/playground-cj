# 恭喜！

你已完成「集合基础」章节的学习！

本章介绍了仓颉三种核心集合类型：
- **ArrayList**：有序、可变长的元素列表，支持按下标访问和增删操作
- **HashSet**：唯一元素集合，自动去重，支持快速成员判断
- **HashMap**：键值对映射，支持按键查找和更新

你还学习了集合迭代、Map-Filter-Reduce 模式、集合转换与排序，以及如何用 HashMap 统计词频。

掌握这些内容后，你可以：
- 选择合适的集合类型处理不同场景的数据
- 自由转换集合类型以适配处理流程
- 实现常见的集合操作算法

接下来的章节将介绍更高级的数据类型和操作，让我们继续探索仓颉的更多特性。
