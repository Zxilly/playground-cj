# 集合排序

排序将集合元素按特定顺序排列。这里在 `ArrayList<Int64>` 上实现选择排序：遍历未排序部分，找到最小元素，交换到正确位置。

算法核心是两层循环：外层循环确定当前位置，内层循环在剩余元素中查找最小值的索引。找到后交换当前位置和最小值位置。完成一轮后，当前位置就是正确的元素。

修改比较条件（将 `<` 改为 `>`），可以实现降序排序。仓颉标准库也提供了更高效的排序函数，这里用基础算法演示排序原理。
