# Sorting

Sorting arranges elements in a predictable order. The example implements selection sort on an `ArrayList<Int64>`.

```cangjie
var smallest = i
if (values[j] < values[smallest]) {
    smallest = j
}
```

The algorithm finds the smallest remaining element and swaps it into place on each pass.

Reverse the `<` comparison to `>` in the inner loop and run -- the output should appear in descending order.
