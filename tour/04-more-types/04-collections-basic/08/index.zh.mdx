# 集合示例

本例演示如何用 `HashMap` 统计标签出现次数。这是"遍历、查找、更新"模式的典型场景。

核心逻辑：对每个标签，先检查 `HashMap` 中是否已存在该键。如果存在，取出当前计数并加一后更新；如果不存在，插入初始值 1。这样遍历完所有标签后，`HashMap` 就记录了每个标签的出现次数。

这种模式在词频统计、投票计数、分组聚合等场景都很常见。`HashMap` 的快速查找特性（平均 O(1)）让这类操作非常高效。
