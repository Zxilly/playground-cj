# Map, Filter, Reduce

Map transforms each element, filter selects elements matching a condition, and reduce combines all elements into a single value.

```cangjie
// map: square each number
squares.add(number * number)
// filter: keep evens
if (number % 2 == 0) { evens.add(number) }
// reduce: sum all
sum += number
```

The three operations are written as explicit loops so the logic is visible step by step.
