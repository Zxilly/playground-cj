# Map、Filter、Reduce

Map、Filter、Reduce 是三种最常用的集合处理模式。这里用显式循环演示它们的核心逻辑。

**映射（Map）** 对每个元素应用转换函数，产生新集合。例如将每个数字平方。**过滤（Filter）** 选出满足条件的元素。例如只保留偶数。**归约（Reduce）** 将所有元素合并为一个值，通常需要一个初始值和累积函数。

仓颉标准库提供了这些操作的高阶函数形式，后续章节会介绍。这里先用显式循环实现，让每一步逻辑清晰可见，适合初学者理解本质。

右侧代码对一个整数列表分别执行：映射（求平方）、过滤（选偶数）、归约（求和）。
