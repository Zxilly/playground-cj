# HashMap

`HashMap<K, V>` maps keys to values with fast key-based lookup.

```cangjie
let scores = HashMap<String, Int64>()
scores.add("alice", 95)
scores["bob"] = 88
```

`add` inserts a pair, subscript `scores[key]` reads or updates a value, `contains` checks for a key, and `remove` deletes an entry. Iteration yields `(key, value)` tuples.

<CompareGroup>

<CompareWith lang="java">
Java's `HashMap<K, V>` uses `put(k, v)` and `get(k)`. Cangjie supports subscript syntax `map[k]` directly, making read/write more concise.
</CompareWith>

<CompareWith lang="go">
Go's built-in `map[K]V` is similar. Access is `m[k]`, deletion uses `delete(m, k)`, and iteration uses `for k, v := range m`.
</CompareWith>

<CompareWith lang="rust">
Rust's `HashMap<K, V>` uses `insert(k, v)` and `get(&k)` (returns `Option`). Cangjie's subscript access is more direct.
</CompareWith>
