# ArrayList

`ArrayList<T>` is a dynamically growing ordered collection where elements are accessed by index. It supports inserting, modifying, and deleting elements at any position.

Unlike `Array`, an `ArrayList` can change size at runtime, making it suitable for scenarios with frequent element additions and removals. Use `add` to append elements at the end, `add(value, at: index)` to insert at a specific position, and `remove(at: index)` to delete elements.
