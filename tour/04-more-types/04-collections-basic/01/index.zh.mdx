# ArrayList

`ArrayList<T>` 是一个可动态增长的有序集合，元素通过下标访问，支持在任意位置插入、修改和删除元素。

与 `Array` 不同，`ArrayList` 的大小可以运行时改变，适合需要频繁增删元素的场景。使用 `add` 在末尾添加元素，`add(value, at: index)` 在指定位置插入，`remove(at: index)` 删除元素。

右侧代码演示了 `ArrayList` 的核心操作：创建空列表、添加元素、插入到指定位置、通过下标读取和修改、删除元素，以及遍历所有元素。

<CompareGroup>

<CompareWith lang="c">
C 需要手动管理内存并重新分配数组大小。仓颉的 `ArrayList` 自动处理内存和扩容。
</CompareWith>

<CompareWith lang="java">
Java 的 `ArrayList<T>` 与仓颉的 `ArrayList<T>` 相似，但命名不同（Java 用 `add`、`remove`、`get`）。两者都提供了动态数组功能。
</CompareWith>

<CompareWith lang="go">
Go 使用切片（slice）实现动态数组，通过 `append` 函数添加元素。仓颉的 `ArrayList` 是专门的类型，提供了 `add`、`insert`、`remove` 等方法。
</CompareWith>

<CompareWith lang="rust">
Rust 的 `Vec<T>` 与仓颉的 `ArrayList<T>` 类似，但 Rust 使用 `push` 和 `remove` 方法。仓颉使用命名参数 `at:` 指定位置。
</CompareWith>

</CompareGroup>
