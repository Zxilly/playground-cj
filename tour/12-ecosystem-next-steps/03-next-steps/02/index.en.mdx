# What's Next

Save an example as a standalone `.cj` file and compile it with `cjc`. When the code grows, use `cjpm init` to start a project.

```cangjie
cjc hello.cj -o hello
cjpm init myproject
```

The tour covered variables, types, control flow, functions, data shapes, classes, generics, error handling, collections, extensions, concurrency, and reflection. Each topic is a building block for real programs.

Pick one example from the tour, save it as a `.cj` file, and compile it locally.
