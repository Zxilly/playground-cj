# 社区资源与进阶学习

深入学习仓颉语言的最佳途径是阅读官方文档和参与社区。仓颉官方文档涵盖了完整的语言规范、标准库 API 和高级特性的详细说明。

GitCode 上托管的[仓颉开源仓库](https://gitcode.com/Cangjie)包含了编译器、标准库的实现源码，是理解语言内部机制的绝佳参考。社区贡献的第三方包由 [Cangjie-TPC](https://gitcode.com/Cangjie-TPC) 组织维护，涵盖网络通信、数据序列化、数据库驱动等多个领域。

参与社区可以让你获得帮助、贡献代码或关注语言发展。无论是提交 Issue、贡献代码，还是分享经验，都是推动生态繁荣的重要方式。

<CompareGroup>

<CompareWith lang="c">
C 的标准由 ISO 维护，官方文档严谨但较难理解。社区资源分散在各个网站上。
</CompareWith>

<CompareWith lang="java">
Java 的官方文档详尽，Oracle 提供了完整的 API 文档。社区非常活跃，第三方库托管在 Maven Central Repository。
</CompareWith>

<CompareWith lang="go">
Go 的官方文档和博客质量极高，社区贡献的包托管在 GitHub 上。仓颉的生态正在建设中，发展潜力巨大。
</CompareWith>

<CompareWith lang="rust">
Rust 有 crates.io 作为中心化的包仓库，文档中心 docs.rs 自动生成 API 文档。仓颉的包管理和文档系统正在向类似方向演进。
</CompareWith>

</CompareGroup>

<Note>
阅读标准库源码是提升编程能力的有效方法。推荐从 `std.collection` 和 `std.io` 这些常用模块开始。
</Note>

访问官网 [cangjie-lang.cn](https://cangjie-lang.cn) 探索更多资源，开始你的进阶之旅！
