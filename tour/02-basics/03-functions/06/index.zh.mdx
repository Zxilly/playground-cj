# 恭喜！

你已经学完了函数这一课。现在你已经掌握了仓颉中函数的核心概念：

- 使用 `func` 关键字定义函数
- 参数列表和返回值类型的基本语法
- 隐式返回让代码更简洁
- 用元组实现多返回值
- 命名参数提高代码可读性
- 默认参数值简化函数调用

函数是程序的构建块，理解函数的定义和使用是进一步学习的基础。在接下来的章节中，你会看到更多函数的应用场景，包括高阶函数、闭包等高级特性。

<CompareWith lang="c">
C 的函数定义相对简单，但缺少许多现代特性。仓颉支持命名参数、默认参数和隐式返回，让函数更易用。
</CompareWith>

<CompareWith lang="java">
Java 的方法需要指定访问修饰符和类型。仓颉的命名参数和默认参数减少了方法重载的需要。
</CompareWith>

<CompareWith lang="go">
Go 与仓颉在函数基础语法上有相似之处，都使用 `func` 关键字。但仓颉支持命名参数和默认参数，这些 Go 需要通过其他方式模拟。
</CompareWith>

<CompareWith lang="rust">
Rust 和仓颉在函数语法上非常相似，都支持隐式返回和元组多返回值。仓颉的命名参数语法 `name!: Type` 是其独特之处。
</CompareWith>
