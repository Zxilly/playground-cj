# 默认参数值

命名参数可以设置默认值，语法为 `name!: Type = defaultValue`。调用函数时，如果省略了有默认值的参数，就自动使用默认值；也可以显式传值来覆盖默认值。

只有命名参数才能有默认值，普通参数不能设置默认值。默认值表达式在函数定义时求值，不是每次调用时重新计算。有了默认参数，可以减少函数重载，让 API 更简洁。
