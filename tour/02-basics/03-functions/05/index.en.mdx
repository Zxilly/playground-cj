# Default Parameter Values

Named parameters can have default values using the syntax `name!: Type = defaultValue`. When calling a function, if you omit a parameter with a default value, the default value is used automatically; you can also explicitly pass a value to override the default.

Only named parameters can have default values; regular parameters cannot have default values. Default value expressions are evaluated at function definition time, not recalculated on each call. Default parameters can reduce function overloading and make APIs cleaner.

The code on the right shows `greet` and `connect` functions, both with default parameters. You can pass only some parameters, while others use default values. Observe the output from different calling styles.

<Note>Only named parameters can have default values. Default values must be compile-time determinable expressions.</Note>

Try calling `connect` passing only the `host` parameter to see what default values are used for port and secure connection.

CJ:
// Define named parameters with default values
func greet(name!: String, greeting!: String = "Hello") {
    println("${greeting}, ${name}!")
}

// Multiple parameters with default values
func connect(host!: String, port!: Int64 = 8080, secure!: Bool = false) {
    let protocol = if (secure) { "https" } else { "http" }
    println("Connecting to ${protocol}://${host}:${port}")
}

// Default values can be computed expressions
func repeat(str!: String, times!: Int64 = 3) {
    var result = ""
    for (i in 0..times) {
        result = result + str
    }
    println(result)
}

main() {
    // Use all default values
    greet(name: "Cangjie")
    
    // Override some default values
    greet(name: "World", greeting: "Welcome")
    println("---")
    
    // Different combinations using default values
    connect(host: "example.com")
    connect(host: "example.com", port: 443)
    connect(host: "example.com", port: 443, secure: true)
    connect(host: "localhost", secure: true)
    println("---")
    
    // Use default times parameter
    repeat(str: "Ha")
    
    // Override default times parameter
    repeat(str: "* ", times: 5)
    
    // All parameters passed explicitly
    repeat(str: "OK ", times: 2)
}
