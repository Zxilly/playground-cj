# 浮点类型

浮点类型用于表示带小数部分的数值，包括 `Float16`、`Float32` 和 `Float64`，分别对应 IEEE 754 的半精度、单精度和双精度格式。`Float64` 是默认的浮点类型，精度约为 15 位有效数字。

浮点字面量支持十进制和十六进制两种形式。十进制字面量必须包含整数部分或小数部分，可以用 `e` 或 `E` 表示指数（底数为 10）。十六进制字面量以 `0x` 开头，必须包含以 `p` 或 `P` 为前缀的指数部分（底数为 2）。

可以通过后缀明确指定类型：`f16`、`f32`、`f64` 分别对应三种浮点类型。右侧代码演示了浮点数的运算和不同表示方式。

<Note>浮点类型不支持自增 `++` 和自减 `--` 操作符，与整数类型不同。</Note>
