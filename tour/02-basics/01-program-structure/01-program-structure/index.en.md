# Program Structure & main

Cangjie programs begin execution from the `main` function. Every executable program must contain a `main` function as its entry point.

The basic form of `main` is straightforward: the function name followed by parentheses `()`, then a function body enclosed in curly braces `{}`. Statements in the body execute sequentially.

If a statement occupies its own line, the trailing semicolon can be omitted. However, if multiple statements appear on the same line, they must be separated by semicolons `;`.

In Cangjie, code enclosed in curly braces `{}` is called a "code block". Expressions in a block execute in order, and the block's value equals the value of its last expression.
