# Comments

Comments are explanatory text in your code that do not affect compilation or execution. Good comments help others (and your future self) understand the code.

Cangjie supports two comment styles:

**Single-line comments**: Start with `//`. Everything from `//` to the end of the line is a comment.

**Multi-line comments**: Start with `/*` and end with `*/`. They can span multiple lines and can be nested.
