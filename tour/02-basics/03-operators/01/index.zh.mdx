# 算术运算符

仓颉提供标准的算术运算符：加（`+`）、减（`-`）、乘（`*`）、除（`/`）、取余（`%`）和求幂（`**`）。这些运算符的行为与你在其他语言中期望的基本一致，但有一些细节需要注意。

整数除法会将结果向零方向舍入。例如，`7 / 3` 得到 `2`，而 `-7 / 3` 得到 `-2`。取余运算 `a % b` 的结果定义为 `a - b * (a / b)`，因此结果的符号与被除数相同。一元负号 `-` 可以对数值取反。

运算符有固定的优先级：一元负号优先级最高，然后是 `**`，接着是 `*`、`/`、`%`，最后是 `+`、`-`。使用括号可以改变运算顺序，让代码意图更清晰。

<CompareWith lang="c">
C 同样没有 `**` 求幂运算符，需要调用 `pow()` 函数。C 对整数溢出的处理较为宽松。
</CompareWith>

<CompareWith lang="java">
Java 也没有 `**` 运算符，需使用 `Math.pow()` 方法。Java 对整数溢出也没有检查。
</CompareWith>

<CompareWith lang="go">
Go 没有求幂运算符 `**`，需使用 `math.Pow` 函数。Go 的取余运算符行为与仓颉相同。
</CompareWith>

<CompareWith lang="rust">
Rust 同样没有 `**` 运算符，使用 `pow()` 方法。Rust 在 debug 模式下会检查整数溢出，而仓颉总是禁止溢出。
</CompareWith>
