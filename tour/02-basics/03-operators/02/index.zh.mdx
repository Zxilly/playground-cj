# 比较与逻辑运算符

比较运算符用于比较两个值，返回 `Bool` 类型的结果。仓颉支持六种比较运算符：相等（`==`）、不等（`!=`）、小于（`<`）、小于等于（`<=`）、大于（`>`）和大于等于（`>=`）。被比较的两个操作数必须是相同类型。

逻辑运算符用于组合布尔值：`&&`（逻辑与）在两个操作数都为 `true` 时返回 `true`；`||`（逻辑或）在任一操作数为 `true` 时返回 `true`；`!`（逻辑非）对布尔值取反。逻辑运算符支持短路求值：对于 `a && b`，如果 `a` 为 `false`，则不会计算 `b`；对于 `a || b`，如果 `a` 为 `true`，则不会计算 `b`。

这些运算符常用于条件判断，组合多个条件形成复杂的逻辑表达式。
