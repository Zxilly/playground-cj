# Strings & Interpolation

The string type in Cangjie is `String`, represented with double quotes `"..."`.

String interpolation uses the `${}` syntax to embed expressions within strings. Any expression can appear inside the braces, and its result will be converted to a string.

Strings support common escape characters: `\n` (newline), `\t` (tab), `\\` (backslash), `\"` (double quote), etc.

Multi-line strings can be written using triple double quotes `"""..."""`.
