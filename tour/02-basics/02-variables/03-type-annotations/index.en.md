# Type Annotations

In Cangjie, you can use type annotations to explicitly specify a variable's type. The annotation is placed after the variable name, separated by a colon `:`.

The syntax is: `let name: Type = value` or `var name: Type = value`.

Type annotations are particularly useful when you need to specify a particular type (e.g., `Int8` instead of the default `Int64`), or when you want to improve code readability.
