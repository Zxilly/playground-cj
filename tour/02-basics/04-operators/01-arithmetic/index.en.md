# Arithmetic Operators

Cangjie supports common arithmetic operators:

- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- `%` remainder
- `**` exponentiation

Integer division truncates the decimal part. For precise division results, use floating-point types.

Cangjie also supports increment `++` and decrement `--` operators, but they can only be used as statements, not as part of an expression.
