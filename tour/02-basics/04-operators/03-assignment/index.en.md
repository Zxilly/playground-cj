# Assignment Operators

Cangjie supports the basic assignment operator `=` and compound assignment operators.

Compound assignment operators combine an arithmetic operation with assignment in a single step:

- `+=` addition assignment
- `-=` subtraction assignment
- `*=` multiplication assignment
- `/=` division assignment
- `%=` remainder assignment
- `**=` exponentiation assignment

Assignment operators can only be used with `var` mutable variables. The type of an assignment expression is `Unit`.
