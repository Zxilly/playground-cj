# 算术运算符

仓颉提供标准的算术运算符：一元负号（`-`）、加（`+`）、减（`-`）、乘（`*`）、除（`/`）、取余（`%`）和求幂（`**`）。这些运算符可用于整数和浮点数类型，其中取余运算只支持整数类型。

对于二元运算符，两个操作数的类型必须相同。整数除法会将结果向 0 方向舍入为整数，例如 `7 / 3` 结果为 `2`，`-7 / 3` 结果为 `-2`。取余运算的结果定义为 `a % b = a - b * (a / b)`，因此结果的符号与被除数相同。

求幂运算符 `**` 用于计算幂次方。当左操作数为 `Int64` 时，右操作数必须为 `UInt64` 类型；当左操作数为 `Float64` 时，右操作数可以是 `Int64` 或 `Float64` 类型。此外，加法运算符还可以用于字符串拼接。
