# Type Annotations

Adding `: Type` after a variable name explicitly annotates the variable's type. Type annotations tell the compiler what type of data this variable should hold.

If type annotation is omitted, the compiler will automatically infer the type based on the initial value. Both approaches have their advantages: explicit annotation makes type relationships clearer, while type inference makes code more concise. When the type context is clear, omitting annotation is recommended.
