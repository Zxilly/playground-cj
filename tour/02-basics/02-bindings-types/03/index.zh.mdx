# 类型标注

在变量名后添加 `: Type` 可以显式标注变量的类型。类型标注告诉编译器这个变量应该持有哪种类型的数据。

如果省略类型标注，编译器会根据初始值自动推断类型。两种方式各有优势：显式标注让类型关系更清晰，类型推断让代码更简洁。在类型上下文明确时，推荐省略标注。
