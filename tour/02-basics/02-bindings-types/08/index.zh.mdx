# Rune 类型

`Rune` 表示单个 Unicode 字符，可以表示 Unicode 字符集中的所有字符，包括中文、表情符号等。字符字面量以 `r` 开头，后跟单引号或双引号包围的字符。

字符字面量有三种形式：单个字符（如 `r'A'`）、转义字符（如 `r'\n'` 表示换行、`r'\t'` 表示制表符、`r'\\'` 表示反斜杠）、通用字符（以 `\u` 开头加花括号中的 1-8 位十六进制数）。`Rune` 可以与 `UInt32` 互相转换，获取或构造字符的 Unicode 码点。

右侧代码展示了各种字符字面量。通用字符 `\u{4f60}` 表示 Unicode 码点为 U+4F60 的字符"你"。试着自己构造一些表情符号的码点，例如 `\u{1F600}` 是笑脸😀。
