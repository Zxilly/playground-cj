# 比较与逻辑运算符

关系运算符包括相等（`==`）、不等（`!=`）、小于（`<`）、小于等于（`<=`）、大于（`>`）、大于等于（`>=`）。这些运算符要求两个操作数类型相同，运算结果的类型为 `Bool`，值为 `true` 或 `false`。

逻辑运算符包括逻辑非（`!`）、逻辑与（`&&`）、逻辑或（`||`）。逻辑与和逻辑或采用短路求值策略：当第一个操作数已经能确定整个表达式的结果时，不会对第二个操作数求值。例如 `false && expr` 直接返回 `false` 而 `expr` 不会被求值。

运算符优先级方面，`&&` 的优先级高于 `||`。在组合复杂条件时，建议使用括号明确逻辑关系，避免因优先级问题导致意外的结果。仓颉的强类型特性保证 `if (a = 5)` 这样的错误代码会在编译时报错。
