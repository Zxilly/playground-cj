# String and Interpolation

`String` represents a text sequence composed of Unicode characters. String literals are enclosed in double or single quotes, and can also use triple quotes `"""` or `'''` for multi-line strings. Multi-line raw strings start with `#`, where escape characters are not interpreted.

Interpolated strings allow embedding expressions within strings using `${expr}` syntax. The curly braces can contain one or more declarations or expressions, and the value of the last item is converted to a string and inserted. This makes string concatenation simple and intuitive.

<CompareWith lang="java">

</CompareWith>

<CompareWith lang="go">
Go uses `fmt.Sprintf` or `+` to concatenate strings, without native string interpolation. Cangjie's `${expr}` syntax is more concise and intuitive.
</CompareWith>

<CompareWith lang="rust">
Rust's `format!` macro uses `{}` placeholders rather than embedded expressions. Cangjie's `${expr}` allows direct expression writing, making it more flexible.
</CompareWith>

</CompareGroup>

<Note>Interpolation expressions can be any expression, including declaration statements. The value of the last item in curly braces is converted to a string.</Note>
