# String 与插值

`String` 表示由 Unicode 字符组成的文本序列。字符串字面量用双引号或单引号包围，还可以用三引号 `"""` 或 `'''` 表示跨越多行的字符串。多行原始字符串用 `#` 开头，转义字符不会被解释。

插值字符串允许在字符串中嵌入表达式，使用 `${expr}` 语法。花括号内可以是一个或多个声明或表达式，最后一项的值会被转换为字符串插入。这使得拼接字符串变得简单直观。

<CompareWith lang="c">

<CompareWith lang="java">
Java 的 `String` 不可变，拼接用 `+` 或 `StringBuilder`；仓颉的插值 `${expr}` 更简洁，避免临时对象。
</CompareWith>

<CompareWith lang="go">
Go 使用 `fmt.Sprintf` 或 `+` 拼接字符串，没有原生字符串插值。仓颉的 `${expr}` 语法更简洁直观。
</CompareWith>

<CompareWith lang="rust">
Rust 的 `format!` 宏使用 `{}` 占位符，而非内嵌表达式。仓颉的 `${expr}` 可以直接写表达式，更灵活。
</CompareWith>

</CompareGroup>

<Note>插值表达式可以是任意表达式，包括声明语句。花括号中最后一项的值会被转换为字符串。</Note>
