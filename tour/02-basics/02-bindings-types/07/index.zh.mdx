# Bool 类型

`Bool` 类型表示逻辑真假，只有两个字面量：`true` 和 `false`。布尔值常用于条件判断和逻辑运算。

仓颉支持三种逻辑运算符：逻辑非 `!`（取反）、逻辑与 `&&`（两者都真才为真）、逻辑或 `||`（有一个为真即为真）。比较运算符（`>`、`<`、`==`、`!=`、`>=`、`<=`）的结果也是 `Bool` 类型。`Bool` 类型不支持算术运算。
