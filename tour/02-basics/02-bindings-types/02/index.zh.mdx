# var（可变绑定）

`var` 声明一个可变绑定，值可以在后续被重新赋值。当你确实需要修改变量的值时，就用 `var`。

可变性是一种选择，不是默认行为。你需要显式写出 `var` 来表明这个变量可能会改变，这有助于读者快速识别代码中哪些地方会修改状态。
