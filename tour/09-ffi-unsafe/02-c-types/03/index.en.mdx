# CFunc

`CFunc<(Params) -> Return>` is the type in Cangjie that represents C function pointers. It corresponds to function pointer types in C and can be used for passing functions between C and Cangjie.

There are three ways to create a `CFunc` in Cangjie. The first is through `foreign` declared functions, which automatically become `CFunc` types. The second is by annotating Cangjie functions with `@C`, making them callable from C code. The third is using lambda expressions marked as `CFunc` type, but note that they cannot capture any variables.

Calling a `CFunc` value must be done within an `unsafe` context because the function pointer may be null or point to an invalid address. The code on the right demonstrates three ways to create and use `CFunc`.
