# CPointer 指针

`CPointer<T>` 是仓颉中表示 C 指针的类型，对应 C 语言中的 `T*`。泛型参数 `T` 必须满足 `CType` 约束，即与 C 语言有对应内存布局的类型。

使用 `CPointer<T>()` 可以创建一个空指针（null），通过 `isNull()` 方法检查指针是否为空。仓颉支持指针算术运算，可以用 `+` 和 `-` 对指针进行偏移。指针的类型转换使用 `CPointer<NewT>(oldPtr)` 形式。

通过指针读取和写入数据是不安全操作，必须在 `unsafe` 块中进行。使用 `read()` 读取指针指向的值，使用 `write()` 写入值。右侧代码演示了分配内存、类型转换、读写数据和指针算术等常见操作。
