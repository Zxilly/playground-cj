# 创建线程（spawn）

仓颉通过 `spawn` 关键字创建轻量级线程（协程）。`spawn` 接受一个代码块，并在新的线程中执行它。

`spawn` 返回一个 `Future` 对象，你可以通过调用 `.get()` 来等待线程完成。如果不调用 `.get()`，主线程可能在子线程完成之前就结束了。

基本语法：
```
let future = spawn {
    // 在新线程中执行的代码
}
future.get()  // 等待线程完成
```

`spawn` 创建的是轻量级线程，开销比操作系统线程小得多，你可以轻松创建成百上千个。右侧的代码演示了如何创建和等待多个线程。
